Celebrity Profiling

CLEF @ PAN 2019

Given the english Social Media feed of a celebrity, determine the degree of fame, occupation, age and gender.

Corpus Download Training Data

The task is to predict four traits of a celebrity from it's social media history. These traits are: 

    fame        := {regional, wellknown, superstar}
    occupation  := {TODO}
    age         := {1,…,114}
    gender      := {male, female, nonbinary}

The social media history consists of the teaser messages of the latest up to 3200 Tweets.

The training dataset contains of two files: a feeds.ndjsonexample and a labels.ndjsonexample. Each file lists all celebrities as json objects, one per line and identified by the cid key. For evaluation, you software will read a feeds.ndjson file from a given directory and write a valid labels.ndjson with your predictions to a given output directory: 

    mySoftware -i path/to/feeds/directory -o path/to/output/directory

Evaluation Scheme Download Evaluation Script

Submissions are judged by a combined metric cRank, which is the harmonic mean of each label's metric. Fame, occupation, and gender are judged by their respective F1 while age is judged by Mean Squared Error.

$$ \text{cRank} = {4 \over {\frac{1}{\text{F}_{1, \text{fame}}} + \frac{1}{\text{F}_{1, \text{occupation}}} + \frac{1}{\text{F}_{1, \text{gender}}} + \frac{1}{\text{MSE}_{\text{age}}}}} $$
