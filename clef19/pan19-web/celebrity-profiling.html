Celebrity Profiling

CLEF @ PAN 2019

Given the english Social Media feed of a celebrity, determine the degree of fame, occupation, age and gender.

Corpus Download Training Data

The dataset consists of two files: a feeds.ndjson and a labels.ndjson. Feeds contains on each line all the tweets of one celebrity. Labels contains on each line the degree of fame, occupation, age, and gender for a celebrity in Feeds. The task is, given only the feeds, to predict the labels accordingly and generate the labels.ndjson. Following values are possible for each trait:

Label Possible values
Fame regional, wellknown, superstar
Occupation TODO
Age numerical age
Gender male, female, other
